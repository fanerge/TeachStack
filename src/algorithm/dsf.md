# 解题的一般步骤

1.  模拟：模拟题目的运行(画图辅助分析)。
2.  规律：尝试总结出题目的一般规律和特点。
3.  匹配：找到符合这些特点的数据结构与算法(匹配消除用栈、找相邻最小最大值用单调栈)。
4.  边界：考虑特殊情况(输入参数的判断等)。

## 大鱼吃小鱼

1.  使用栈匹配消除（消除小鱼）


## 单调栈的解题技巧

1.  递增栈，栈元素是递增的（消除，较小数消除较大数）
2.  递减栈，栈元素是递减的（消除，较大数消除较小数）
